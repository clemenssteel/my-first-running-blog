Datensätze

Es liegen 88 Datensätze verschiedener Aktivitäten vor. Diese enthalten den kontinuierlichen Puls, die gelaufenen Meter und die kontnuierliche Position als Geokoordinaten, außerdem Maximalpuls, mittleren Puls und den Startzeitpunkt.

publiziert: 5. Oktober 2018 9:14

Datenverarbeitung

Die Daten liegen im Garmin Dateiformat vor und können als xml Datei geparst werden. Eine Run Klasse erzeugt Run Objekte mit dem Startzeitpunkt als Namen und den Daten als array.
