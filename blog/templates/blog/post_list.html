Running Blog zum Thema Laufdatenauswertung

Es wird mit Pulsgurt aufgezeichnet und per App an GoogleFit gesendet.

Die Google Fit Daten werden mit Garmin Schnittstelle heruntergeladen und mit Python ausgewertert.

GitHub repository ohne Laufdaten: https://github.com/clemenssteel/TrainingCenter

Stand des Arbeitsfortschritts am Projekt: https://trello.com/b/VlcS8lx9/trainingcenter

publiziert: 05.10.2018, 08:15:

Datensätze

Es liegen 88 Datensätze verschiedener Aktivitäten vor. Diese enthalten den kontinuierlichen Puls, die gelaufenen Meter und die kontnuierliche Position als Geokoordinaten. Außerdem Maximalpuls, mittlerer Puls und der Startzeitpunkt.

published: 05.10.2018, 09:14

Datenverarbeitung

Die Daten liegen im Garmin Dateiformat vor und können als xml Datei geparst werden. Eine Run Klasse erzeugt run Objekte mit dem Startzeitpunkt als Namen und den Daten als Dictionary mit Listen gefüllt.
